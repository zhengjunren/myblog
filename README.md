# myblog

## 介绍
该博客系统是我大二上学期期末作业修改而来，目前是 springboot 项目。

## 软件架构
目前仍然是单体

分为`common`、`admin`和`frontend`，其中前端采用的是 [`vue-admin-template`](https://github.com/PanJiaChen/vue-admin-template)模板。


## 安装教程
### redis
启动命令：

```redis-server redis.windows.conf```
