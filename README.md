# myblog

#### 介绍
该博客系统是我大二上学期期末作业修改而来，目前是 springboot 项目。

#### 软件架构
目前仍然是单体

分为`business`、`security`和`frontend`，其中前端采用的是 [`vue-admin-template`](https://github.com/PanJiaChen/vue-admin-template)模板。
`business`是资源服务器，即业务层；`security`为认证服务器。
